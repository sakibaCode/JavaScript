Array:

  In JavaScript, an array is an ordered list of values. 
  Each value, known as an element, is assigned a numeric position in the array called its index.
  The indexing starts at 0, so the first element is at position 0, the second at position 1, and so on.

Creating Array:

    // Creating an Empty Array
    let a = [];
    console.log(a);

    // Creating an Array and Initializing with Values
    let b = [10, 20, 30];
    console.log(b);

    // Creating and Initializing an array with values
    let a = new Array(10, 20, 30);

    console.log(a);

Accessing Array Element:

        // Accessing First Array Elements
        let fst = a[0];

        console.log("First Item: ", fst);

        // Accessing Last Array Elements
        let lst = a[a.length - 1];

        console.log("First Item: ", lst);
